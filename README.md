Project: Wacky’s Restaurant – CSCA08, Fall 2024

This project simulates a small restaurant system, handling customer seating, menu orders, and cost/profit calculations.


Key features:

Manage individual and group seating, including seat selection and cleaning

Validate menu items and combos

Compute item and combo prices, costs, and profits

Parse customer requests from natural-language sentences


Skills demonstrated:

String manipulation and parsing

Conditional logic and data validation

Modular function design with clear preconditions and outputs

Basic computational modeling (price, cost, profit)
